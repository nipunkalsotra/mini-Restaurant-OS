# 🍽️ Mini Restaurant OS

A full-stack Restaurant Management System built using **React + FastAPI**.

## 🚀 Features

* Menu browsing by restaurant
* Cart system
* Customer selection & creation
* Billing system (paid & unpaid)
* Order lifecycle management
* Payment handling
* Sales tracking

## 🛠️ Tech Stack

* Frontend: React
* Backend: FastAPI
* Database: PostgreSQL
* ORM: SQLAlchemy

## ⚙️ Setup (Local)

### Backend

```bash
cd Backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## 🌐 Deployment

* Frontend: Vercel
* Backend: Render
* Database: Neon (PostgreSQL)
